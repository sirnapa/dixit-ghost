Echo Documentation

Thanks for purchasing Echo, a responsive and minimalistic theme for Ghost. It is designed to focus on your content and present it beautifully.

Here you will be able to find out how to install and customize the theme.

1 . Getting Started

For the purposes of this documentation, it is assumed that you have Ghost up and running. If not, please visit docs.ghost.org and follow the guide to install and configure Ghost.

2. Installation

  1. Navigate to the directory on your server where Ghost is installed.
  2. Once there, navigate to the /content/themes/ directory.
  3. Upload the theme files to this directory.
  4. Restart Ghost.
  5. Log into your Ghost Admin.
  6. Navigate to Settings > General and select Echo in the theme options dropdown menu.
  7. Click 'Save'.

Note: If you're on the Ghost Hosted Service, rather than a self-install, to switch theme you'll need to go to your blog management page and click on "edit" beside the name of your blog.

3. Settings

In Settings > General several options will alter the content in Echo:

  • Blog Logo - Appears as the first item the header.
  • Blog Title: - Appears beside the blog logo in the header.
  • Blog Description - Appears as the first item in the sidebar.
  • Blog Cover - This will appear as a full background image in the underneath the header.
  • Posts per page - This is how many posts are displayed per page.

4. Customization

If you're comfortable with HTML & CSS, you can make any changes to the theme by editing Echo's template files.

Please be aware that when Echo is updated in the future, you will need to re-integrate your custom theme changes.

The CSS files were created using SASS. If you are familiar with SASS, you can easily customize Echo.

Using CSS

If you don't want to use SASS, you can edit the Echo's main CSS file (screen.css). This file is located in the /assets/css/ directory.

The best way to change the colors in Echo is to use the Find & Replace feature found in many popular text editors.

Here are the colors you should look for:

  • Primary Color: #30363c (Black)
  • Secondary Color: #5c6872 (Dark Grey)
  • Tertiary Color: #929aa0 (Light Grey)
  • Font Color: #535e67 (Dark Grey)
  • Accent Color: #DD574C (Red)
  • Border Color: #d1d6da (Light Grey)

Using SASS

You will find the following SCSS files in /assets/scss/ directory:

  • mixins.scss - This contains all the CSS declarations that Echo reuses throughout the theme.
  • screen.scss - This is the main CSS stylesheet. It imports the mixins.scss and variables.scss files.
  • variables.scss - This contains all the variables that you can tweak to customize the way Echo looks.

Note: You will need a SASS preprocessor, such as CodeKit (Mac only) or LiveReload to use the SCSS files.

Navigation

You can customize the navigation menu by editing the navigation.hbs file located in the partials directory.

This is a great place to put links to your static pages or custom links to your website.

  1. Navigate to the partials directory on your server.
  2. Open the navigation.hbs file in your favorite text editor.
  3. Replace the hashtags (#) with your own links.
  4. Upload the changed navigation.hbs file to the partials directory.

Example With Absolute Links:

<li>
    <a href="http://www.your-website-here.com">About Me</a>
</li>
<li>
    <a href="http://www.your-website-here.com">Resume</a>
</li>

Example With Static Page Links:

<li>
    <a href="about">About Me</a>
</li>
<li>
    <a href="resume">Resume</a>
</li>

Favicon

You can add you own custom Favicon to your blog by replacing the favicon.ico file located in the assets directory:

  1. Navigate to the assets directory on your server.
  2. Replace the favicon.ico file with your own custom icon.

There are many different web based tools for converting an image into a favicon.ico file.

Cover Photo Text

You can add custom text to appear on top of the cover photo by editing the cover-photo.hbs file located in the partials directory.

This is a great place to put your blog's tagline or motto.

  1. Navigate to the partials directory on your server.
  2. Open the cover-photo.hbs file in your favorite text editor.
  3. Replace the text between the h2 HTML tags.
  4. Upload the changed cover-photo.hbs file to the partials directory.

The cover photo text will only appear in you upload a cover photo image to your Ghost blog.

Featured Images

You can set a featured image by placing it as the first thing in your post. This featured image will be displayed along with an excerpt of the post on the homepage.

  1. Create or edit a post.
  2. Insert an image as the first thing in the post.
  3. Save/update the post.

This feature is not limited to images, you can also feature YouTube, Vimeo or SoundCloud embeds.
